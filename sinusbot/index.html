Anleitung:
Lade dieses Script herunter und verschiebe es per FTP-Zugang in einen Ordner.
Danach machst du dieses Script ausführbar (chmod u+x /pfad/zum/Script).
Nun führst du das Script aus (./scriptname).

Aktuell wird Debian8, Ubuntu 14.04 LTS und höher, sowie CentOS 7 unterstützt.

- Ausgeführt wird folgendes:
- Check ob Linux-Version (Debian, Ubuntu, CentOS) unterstützt wird
- Installieren des TeamSpeak-Clients (neuste Version)
- Installieren von allen notwendigen Paketen
- Erstellen eines seperaten Users für den Bot
- Installieren des Beta Bots
- Installieren des YouTube-DL
- Cronjob für tägliches Update des YouTube-DL und Sinusbot (sobald unterstützt)
- Anpassen aller Rechten
- Erstellen systemd Datei => starten/stoppen über "service sinusbot {start|etc}"
- Selbstständiges löschen aller ungebrauchten Dateien und Archive
- Startet den Sinusbot im Anschluss an die Installation
- Auf Wunsch Update und Entfernung des Bots

Der Installationsprozess dauert circa eine Minute.
